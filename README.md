## How to run this project

### Requirements:

-   node
-   npm
-   yarn

#### Install packages:

```shell
$ yarn
```

#### Add your backend api url to .env file:

```text
VITE_API_BASE_URL="API_BASE_URL"
```

#### Run the project:

```shell
$ yarn dev
```
